
Customizable organizer tray with parameters for:
   - Size, number, and configuration of slots
   - Size, shape, and number of oversized slots
   - Inner and outer wall thickness
   - Optional relief holes on sides, bottom, or both
   - Rounded or sharp edges
   - Optional lid.  (For now this is just a simple detached lid.  I plan to add hinged and/or latching lids to later updates.)
